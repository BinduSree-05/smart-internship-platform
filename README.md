# 🚀 AI-Powered Smart Internship Platform

A basic full-stack MERN application that helps students discover and apply for internships based on their skills and resume analysis.

---

## 🌟 Features

* 🔐 JWT Authentication (Login/Register)
* 📄 Resume Upload & Skill Extraction
* 🤖 Skill Match Percentage Calculation
* 🏢 Internship Listings
* ✅ Internship Application System
* ☁️ MongoDB Atlas Integration
* 🌐 Frontend deployed on Vercel
* ⚡ Backend deployed on Render

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Axios
* Tailwind CSS

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Multer
* PDF/Text Resume Parsing

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/BinduSree-05/smart-internship-platform.git
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
node server.js
```
